Arduino-based Clock System

The Arduino-based Clock System is a real-time clock (RTC) application built with an Arduino Mega and a DS1320 RTC module. This project features an LCD 16x2 display to show the current time and date, with the ability to set the time and display it accurately. The system leverages an RTC module to maintain accurate timekeeping even when the power is turned off. This project demonstrates my skills in embedded systems and working with microcontrollers.
